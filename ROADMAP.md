### 0.1 Что ещё нужно получить

**BOT_TOKEN (новый):**
1. Напиши @BotFather → `/mybots`
2. Выбери своего бота
3. `API Token` → `Revoke current token`
4. Скопируй новый токен → вставь в `.env`

**ADMIN_IDS:**
1. Напиши @userinfobot в Telegram
2. Скопируй числовой ID → вставь в `.env` как `ADMIN_IDS=123456789`

**SPREADSHEET_ID:**
1. Открой drive.google.com → создай новую таблицу Google Sheets
2. Назови "Астана Аренда"
3. Из URL скопируй ID между `/d/` и `/edit`
   Пример: `https://docs.google.com/spreadsheets/d/`**`1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms`**`/edit`
4. Вставь в `.env` как `SPREADSHEET_ID=...`

**service_account.json:**
1. Открой console.cloud.google.com
2. Выбери проект `gen-lang-client-0940989557`
3. APIs & Services → включи: Google Sheets API + Google Drive API
4. Credentials → Create credentials → Service Account
5. Скачай JSON → положи в `secrets/service_account.json`
6. Открой таблицу → Поделиться → вставь `client_email` из JSON → Редактор

**CLAUDE_API_KEY:**
1. Открой console.anthropic.com
2. API Keys → Create Key
3. Вставь в `.env`

**GitHub PAT:**
1. Открой github.com/settings/tokens/new
2. Note: `astana-bot-deploy`
3. Expiration: No expiration
4. Scopes: поставь галочку `repo`
5. Нажми Generate token → скопируй токен (начинается с `ghp_`)
6. Сохрани в терминале одной командой (Windows):
   ```
   echo "https://ВАШ_ЛОГИН:ВАШ_ТОКЕН@github.com" >> %USERPROFILE%\.git-credentials
   ```
