## Как привлекаем клиентов — 3 канала

### Канал 1: Telegram парсинг
Python скрипт (Telethon) сканирует 20+ чатов каждые 30 минут → находит сообщения с ключевыми словами (ищу SMM, нужен маркетолог, продвижение) → Gemini AI пишет персональное первое сообщение → отправляется автоматически → лид попадает в CRM.
**Лимит:** 30–50 сообщений в день

### Канал 2: 2ГИС + Instagram
n8n делает HTTP запрос к 2ГИС API → парсит владельцев бизнеса по нишам → рассылка предложения. Phantombuster парсит Instagram по хэштегам → авто-DM.
**Лимит:** 20–30 контактов в день

### Канал 3: Таргетированная реклама
Meta Ads + TikTok Ads → лид-форма с оффером мини-курса или бесплатной консультации → Webhook → n8n → CRM → SendPulse бот прогревает автоматически.
**Бюджет:** $150–500/мес

## Что делаем для клиентов на продюсировании

1. Записываем подкаст с клиентом раз в месяц (3–5 часов материала)
2. Vizard нарезает на 10–15 Reels автоматически — субтитры, хайлайты, 9:16
3. Gemini AI пишет подписи, хэштеги, описания в голосе клиента
4. Публикации выходят по расписанию каждые 2 дня — 3–4 недели вперёд
5. Запускаем рекламу: 9 креативов, 3 аудитории, A/B тест
6. SendPulse бот прогревает подписчиков → автоматически продаёт курс/консультацию
7. Еженедельный анализ метрик → корректировка стратегии
8. Ежемесячный отчёт клиенту + план на следующий месяц
