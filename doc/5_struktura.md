## 4. CRM Пайплайн — 6 этапов

```
🔵 Новый → 📤 Сообщение → 💬 Ответил → 🔥 Заинтересован → 📞 Созвон → 📝 Контракт
```

| Этап | ID | Цвет | Следующий триггер |
|------|----|----|------------------|
| 🔵 Новый | `new` | #6366F1 | Отправить первое сообщение (AI) |
| 📤 Сообщение | `contacted` | #06B6D4 | Ждать ответа, follow-up 24ч |
| 💬 Ответил | `replied` | #F59E0B | Квалифицировать, предложить созвон |
| 🔥 Заинтересован | `interested` | #EC4899 | Отправить ссылку Calendly |
| 📞 Созвон | `call` | #8B5CF6 | После созвона → договор или архив |
| 📝 Контракт | `contract` | #22C55E | Создать клиента → старт карты бизнеса |

## 6. Контент-потоки — схема

```
ПОТОК 1: Подкаст → Reels (основной)
─────────────────────────────────────
Директор снимает подкаст (3–5ч) 1×/мес
   ↓
Google Drive (папка /Подкасты/)
   ↓
n8n Watch для Google Drive (триггер: новый файл)
   ↓
POST /project/create → Vizard API
   ↓
n8n Poll (каждые 5 мин пока code=1000)
   ↓
code=2000 → получить clips[] с viralScore
   ↓
Для каждого клипа: POST /project/ai-social → подпись
   ↓
POST /project/publish-video (publishTime +2 дня за клип)
   ↓
✅ 15 Reels запланированы на 1 месяц вперёд
─────────────────────────────────────
Участие директора: 0 минут после съёмки

ПОТОК 2: Архив видео → Buffer
─────────────────────────────────────
Старые видео добавляются в платформу с URL
   ↓
Gemini AI → заголовок + описание + CTA + хэштеги
   ↓
Buffer API: POST /updates/create (scheduled_at)
   ↓
✅ Посты выходят каждые 3 дня
─────────────────────────────────────

ПОТОК 3: Рекламные креативы
─────────────────────────────────────
Gemini AI → 9 текстов (3 группы × 3 варианта)
   Группа 1: боль → решение (холодная аудитория)
   Группа 2: желание → результат (тёплая)
   Группа 3: социальное доказательство (ретаргет)
   ↓
Meta Ads Manager → загрузить → A/B тест
   ↓
n8n еженедельно → анализ → отключить слабые
─────────────────────────────────────
```

## 7. n8n Воркфлоу — структура

| Воркфлоу | Триггер | Узлы | Выход |
|----------|---------|------|-------|
| 1. TG Parser | Cron каждые 30 мин | Telethon → Filter → Gemini → TG API → HTTP | Лид в CRM |
| 2. Drive→Vizard | New File in Drive | Drive → Vizard Create → Poll → Gemini → Publish | Reels в расписании |
| 3. Meta Webhook | Webhook POST | Parse → HTTP CRM → SendPulse → Gemini → TG | Лид + прогрев |
| 4. Analytics | Cron воскресенье | Meta API → IG API → Gemini → TG Bot → HTTP | Недельный отчёт |
| 5. Archive→Buffer | Cron/Manual | HTTP PC → Gemini → Buffer API → HTTP | Посты запланированы |
