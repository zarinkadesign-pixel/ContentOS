## Критерии приёмки

| Модуль | Критерий | Метод |
|--------|---------|-------|
| CRM | Лид проходит все 6 этапов | Ручной тест |
| AI Агент | Результат за < 30 сек | DevTools Network |
| Vizard | Видео нарезается и публикуется | Реальный тест |
| Финансы | KPI обновляется при транзакции | Добавить → проверить % |
| Карта бизнеса | Чеклист обновляет прогресс | 5 галочек → проверить % |
| Мобильность | Полная функциональность на телефоне | iPhone Safari + Android Chrome |
| Данные | Не теряются при перезагрузке | Refresh → данные на месте |
| Экспорт | JSON содержит все поля | Экспорт → открыть JSON |
